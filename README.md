## Getting Started

To Run the application locally-
1. Clone the repository 
    <br>
    `git clone https://github.com/Shaurya-Saini/Priv_Guard.git`<br>
    `cd Priv_Guard`
2. Create project
    <br>
    `flutter create .`
3. Run the application from `main.dart`
